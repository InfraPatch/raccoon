### Introduction

As an administrator you are able to create contract templates, and new Property Categories.  
You are also able to declare new administrators, and lawyers.

![adminPanel.png](/images/en/admin/adminPanel.png)

### Creating a new contract template

To create a new contract template, all you have to do is give it a name, a
description, and upload the **.docx** file of your template.
Optionally, you can also choose a Property Type.

![newContract.png](/images/en/admin/newContract.png)

**.docx** template creation is discussed in a separate guide.

Property Category is discussed in a separate guide.
