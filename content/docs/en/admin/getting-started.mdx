### Introduction  
As an administrator you are able to create contract templates.  
You are also able to declare new administrators.  
![adminPanel.png](/images/en/admin/adminPanel.png)  

### Creating a new contract template  
To create a new contract template, all you have to do is give it a name, a description, and upload the **.docx** file of your template:  
![newContract.png](/images/en/admin/newContract.png)  

**.docx** template creation is discussed in a separate guide.
